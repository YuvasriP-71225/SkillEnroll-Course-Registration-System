SkillEnroll Course Registration System

Project Overview

SkillEnroll Course Registration System is a full-stack web application built with Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.
It allows students to register for courses, view available courses, and manage enrollments through a simple and interactive interface.
The system ensures data integrity by preventing duplicate or multiple course enrollments.

Features

1. Register students with name, email, and course selection
2. Display all available courses dynamically
3. Prevent duplicate enrollments with SweetAlert pop-ups
4. Unenroll students with confirmation alerts
5. Integrate backend (Spring Boot REST APIs) with frontend (HTML & JS) using Fetch API
6. Maintain student and course data in MySQL database

Tech Stack

Backend: Java, Spring Boot, Spring Data JPA, RESTful API
Frontend: HTML, CSS, JavaScript
Database: MySQL
Build Tool: Maven
Development Environment: Eclipse IDE
